# AnimeTerm

A lightweight, terminal-styled utility for searching and accessing anime content.

## Overview

AnimeTerm provides a streamlined desktop interface for browsing the Jikan (MyAnimeList) database. It features a high-contrast dark theme and intelligent routing to external streaming and download services.

## Features

- **Search**: Instant access to anime metadata via Jikan API.
- **Watch**: Automatic redirection to stream episode 1 on AnimeAV1.
- **Download**: Quick search integration with Shana Project for torrents.
- **Legal Streams**: Shows available platforms (Crunchyroll, Netflix, etc.) directly in the detail window.
- **Visuals**: "Blue Matrix" aesthetic using standard Tkinter libraries.

## Usage

Ensure you have Python 3 installed.

1. Install dependencies:
   ```bash
   pip install requests
   ```

2. Run the application:
   ```bash
   python animeterm.py
   ```

Or download `AnimeSearch.exe` from [Releases](https://github.com/lordvamp9/Anime-Search/releases).
